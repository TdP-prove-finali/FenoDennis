Analisi del Sistema GTFS
########################

Istruzioni per l'installazione e l'utilizzo dell'applicazione.

1) Visionare il video dimostrativo di utilizzo dell’applicazione, al seguente indirizzo:

***	https://youtu.be/HxUIT9YBhCo

2) Scaricare il progetto dalla pagina di GitHub:

***	https://github.com/TdP-prove-finali/FenoDennis

3) Importare sul proprio DB server il database dell’applicazione, 
presente all’interno della cartella “database” del progetto.

4) Lanciare il file “Analisi_Sistema_GTFS_Feno_Dennis”,
presente all’interno della cartella “app” del progetto di GitHub.

In alternativa al passo 4) Aprire il progetto scaricato con Eclipse e 
lanciare la classe Main nella directory principale del codice sorgente.


Requisiti: 
.Java 1.8 o successivo
.MySQL 5.5 o successivo
